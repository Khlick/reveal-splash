Installation Instructions

To install the Splash plugin for Reveal.js, you can either include it directly in your HTML file using a module script or install it via npm if you're using a Node.js environment.

Option 1: Direct HTML Include

Option 2: Install via npm


            npm install reveal-splash

After installation, import the plugin in your JavaScript file:


            import Splash from 'reveal-splash';

Usage

Once the plugin is installed, you can initialize your Reveal.js presentation with the Splash plugin by adding it to the Reveal.initialize call.


            Reveal.initialize({
                plugins: [Splash]
            });

Configuration Options

The Splash plugin provides several configuration options that allow you to customize the splash screen. Below is a table describing each option, its type, default value, and description:

Option Type Default Description
splashImage Boolean/String false Specifies the image to display on the splash screen. If false, a spinner will be used.
backgroundColor String "#ffffff" The background color of the splash screen. Accepts any valid CSS color.
text String "Loading..." The text to display on the splash screen. Can be a plain string or HTML.
fontOptions Object { fontFamily: "Arial, sans-serif", fontSize: "18px", color: "#333" } An object defining the font properties of the splash text. Supports standard CSS font properties.
minimumDisplay Number 1 The minimum time (in seconds) that the splash screen should be displayed.
